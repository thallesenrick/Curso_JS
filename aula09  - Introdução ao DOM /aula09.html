Aprendendo a usar o DOM em JAVASCRIPT

DOM é Document Object Model

Árvore DOM começa no window dentro de window nos temos o LOCATION: diz qual a localização do site, em qual url está, qual a pagina atual, qual a anterior

Depois tem o DOCUMENT: que trata sobre o documento atual. Dentro do document tem o HTML. Dentro do HTML tem o HEAD e o BODY

Depois tem o HISTORY: que vai guardando de onde vc veio para onde vc vai
